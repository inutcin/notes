Rdp (подключение к удалённому рабочему столу)
=====================================


# Установка RPD-сервера

На машине куда подключаемся

Устанавливаем пакеты

```
$ sudo apt install xrdp  xorgxrdp
```

Добавляем пользователя в группу для чтения сертификатов

```
$ sudo adduser xrdp ssl-cert
```

Перезапускаем rdp-сервер

```
$ sudo systemctl restart xrdp
```

# Установка RPD-клиента

```
$ sudo apt install freerdp2-x11
```

# Подключение к удалённому серверу

```
$ xfreerdp -u r-asian -p ******* -g 1024x768 10.240.2.19
```
