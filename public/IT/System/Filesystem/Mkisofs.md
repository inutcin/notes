Создание образа CD/DVD
========================


# Образ CD/DVD диска из указанного каталога

Создать образ **output.iso** из каталога **my_files** с поддержкой Linux и
Windows длинных имён файлов

```
$ mkisofs -o output.iso -J -R my_files/
```

- **-J** - использовать Joliet расширение для поддержки длинных имён файлов в Windows
- **-R** - использовать Rock Ridge расширение для поддержки длинных имён фалов в Uni-подобных ОС
